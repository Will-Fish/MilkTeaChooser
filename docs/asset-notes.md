# 奶茶/咖啡网站素材说明

更新时间：2026-06-18

## 已整理内容

- `data/brands.json`：首批 26 个常见奶茶/咖啡连锁品牌，包含品牌定位、品类、代表单品、分组 SKU、视觉关键词和图片检索词。
- `data/image-sources.csv`：图片来源入口与授权状态。当前均按 `needs-permission` 标注，适合作为人工核对、采购授权或替换素材的清单。
- `public/images/`：预留给后续本地授权图片。建议文件命名使用 `brand-id/item-slug.jpg`。

## 图片使用建议

品牌官网、公众号、电商平台、新闻稿中的产品图通常不是可直接商用素材。网站上线前建议采用以下方式之一：

1. 向品牌方或图库购买/申请授权。
2. 自行购买饮品拍摄，避开明显商标特写时可作为“饮品风格图”使用。
3. 使用生成式图片做非品牌化类目图，例如“葡萄芝士茶”“生椰拿铁”，但不要标成某品牌官方产品图。
4. 前端先接入 `imageSearchQueries` 或 `source_url` 做素材管理后台，等图片确认后再落本地。

## 数据字段建议

前端可优先读取：

- `nameZh`：展示名称。
- `type`：`tea` 或 `coffee`，用于筛选。
- `categories`：品类标签。
- `representativeItems`：推荐卡片、搜索词和初始内容。
- `skus`：按品类分组的 SKU 列表，适合品牌详情页、筛选页和搜索联想。
- `visualKeywords`：生成类目图或做设计风格参考。
- `imageSearchQueries`：素材补采关键词。

## SKU 口径

`skus` 是扩展网站素材库口径，目标是覆盖品牌常见产品线、代表饮品、风味变体、烘焙/轻食或周边，方便做页面、推荐、筛选和搜索。它不等同于品牌实时完整菜单；不同城市、门店、季节限定、区域限定会变化。需要上线做交易或价格展示时，建议再接入门店级菜单源。
